A Task describes a job and has the following properties:

  • The ID is used for unique identification of a Task. It is set automatically and is not editable.
  • The parent-ID is used to associate SubTasks to their parent Task.
  • The name briefly describes the task.
  • The dueDate is the date the task is due to be completed.
  • The priority describes how important the Task is. Available priorities are 'low', 'medium' and 'high'.
  • The Progress shows how much of the task is done already.
  • The description is used for a detailed description of the task.
